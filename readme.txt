This assignminet consists of 3 timers of 3 hours, half hour and 1 hour respectively.
When clicked on "Click to START" the timer starts. The start button becomes invisible and a reset button is seen.
When clicked on "RESET" button, the timer goes to original position i.e shows the "Click to START" button. 
The start button is again visible and when click will start the timer again from start.
